Description

The player start in the cell stage, the task is to evolve into sentient specie, advance stages, eras toward the intergalactic era. Under the advance the player has to watch to the resources, such as food. The game has wonders, thats give a high bonus. Have an SQL database to save the progress for players.

Game mechanics

Resources management

  • Raw materials for manufactured goods
  • Storage system

Economy

  • Money changes through eras
  • Banking

Wonders

  • "Special projects" in eras
  • High bonus
  • Expensive build

Military

  • Troops to train
  • diferrent type of warfare

Space colonize

  • Explore star systems
  • Build colony on planets
