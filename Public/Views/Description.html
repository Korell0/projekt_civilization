Description

The players start in the cell stage. The task is to evolve into a sentient species, advance stages and eras towards the intergalactic era. During the advancement, the players have to watch the resources, such as food, wood or uranium. They can build wonders, which provide high bonuses. The game has an SQL database to save the players' progression.

Game mechanics

Resource management

  • Raw materials for manufactured goods
  • Storage system

Economy

  • Money changes through eras
  • Banking

Wonders

  • "Special projects" in eras
  • High bonuses
  • Expensive to build

Military

  • Troops to train
  • Different types of warfare

Space colonization

  • Exploring star systems
  • Building colonies on planets
  • Terraforming planets
